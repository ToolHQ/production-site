Quickstart

Get your first AI cost event in under 60 seconds. No credit card, no SDK lock-in.

60-second time-to-first-event

Add to your IDE's MCP configRecommended

Works with Cursor, VS Code Copilot, OpenCode, Claude Code, and any MCP-compatible IDE.

{
  "mcpServers": {
    "agent-meter": {
      "command": "agent-meter-mcp-wrapper",
      "args": ["--endpoint", "https://agent-meter.dnor.io"]
    }
  }
}

Install the wrapper binary

Download from GitHub releases or build from source:

cargo install agent-meter-mcp-wrapper

Use your IDE normally

Every tool call gets metered automatically. Check your Dashboard for live data.

Install agent-meter-proxy

Single binary that intercepts IDE ↔ LLM HTTPS traffic. Zero code changes.

cargo install agent-meter-proxy

Start the proxy

agent-meter-proxy --otlp-endpoint https://agent-meter.dnor.io:4318

Point your IDE to the proxy

Set HTTPS_PROXY=http://127.0.0.1:9090 in your environment or IDE settings.

export HTTPS_PROXY=http://127.0.0.1:9090

Enable VS Code OTLP telemetry

VS Code Copilot 1.250+ natively exports OTLP spans. Add to settings.json:

{
  "github.copilot.advanced": {
    "telemetryEndpoint": "https://agent-meter.dnor.io:4318/v1/traces",
    "captureContent": true
  }
}

Done — no binary needed

Copilot sends trace data directly. Check Conversations within seconds.

Add agent-meter MCP to CursorZero code

Add to .cursor/mcp.json in your project root:

{
  "mcpServers": {
    "agent-meter": {
      "command": "agent-meter-mcp-wrapper",
      "args": ["--endpoint", "https://agent-meter.dnor.io"]
    }
  }
}

Install the wrapper

cargo install agent-meter-mcp-wrapper

Code with Cursor normally

Every Composer / Cmd+K / Ctrl+I call is metered. See real-time cost on the Cost page.

Add to Claude Code's MCP configZero code

Run in your terminal:

claude mcp add agent-meter -- agent-meter-mcp-wrapper \
  --endpoint https://agent-meter.dnor.io

Install the wrapper

cargo install agent-meter-mcp-wrapper

Use Claude Code

All tool calls (read_file, edit, bash, etc.) are tracked with timing and token counts. Check Conversations.

Install the package

pip install agent-meter

Instrument your code

from agent_meter import AgentMeter

am = AgentMeter(api_key="am_live_...")

span = am.track("web_search", model="gpt-4o")
result = search(query)
span.finish()

Run your app

Spans flush automatically every 5s. View them on the Dashboard in real time.

Install the package

npm install @agent-meter/sdk

Instrument your code

import { AgentMeter } from "@agent-meter/sdk";

const am = new AgentMeter({ apiKey: "am_live_..." });

const span = am.track("web_search", { model: "gpt-4o" });
const result = await search(query);
am.finish(span);

Run your app

Spans flush automatically every 5s. View them on the Dashboard in real time.
