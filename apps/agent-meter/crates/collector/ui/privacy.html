Privacy Policy

Last updated: June 8, 2026

1. What We Collect

Telemetry data (via OTLP / MCP / Proxy)

  • Tool call names, durations, timestamps
  • Model names, token counts (input/output/cached)
  • Conversation IDs, agent identifiers, IDE type
  • Optionally: prompt text and response previews (if captureContent is enabled by the user)

Account data (via GitHub OAuth)

  • GitHub username, email, avatar URL
  • Organization membership (for team features)

Usage metadata

  • IP address (for rate limiting; not stored long-term)
  • Browser user-agent (for compatibility)

2. How We Use It

  • Display dashboards, cost reports, and alerts to you.
  • Compute USD cost attribution per call.
  • Enforce budget limits and fire alert notifications.
  • Improve the Service (aggregate, anonymized analytics only).

3. Data Storage & Retention

  • All data is stored in PostgreSQL within our self-hosted infrastructure (OCI ARM64 cluster).
  • Telemetry events are retained for 90 days by default (configurable per plan).
  • Account data is retained until you delete your account.
  • We do not use third-party analytics, ad networks, or tracking pixels.

4. Data Sharing

We do not sell, rent, or share your data with third parties. Exceptions:

  • If required by law (court order, subpoena).
  • To protect our rights or prevent fraud.

5. Security

  • All traffic is encrypted via TLS (HTTPS).
  • API keys are stored as SHA-256 hashes.
  • Database access is restricted to the application layer.
  • Infrastructure runs on dedicated nodes (not shared cloud tenants).

6. Your Rights

  • Access: Request a copy of your data via API (GET /api/export/traces).
  • Deletion: Request account and data deletion by contacting support.
  • Portability: Export your data in OTLP JSON format at any time.
  • Opt-out: Disable captureContent to stop sending prompt/response text.

7. Cookies

We use a single session cookie (am_session) for authentication. It is HttpOnly, Secure, and SameSite=Lax. No tracking cookies.

8. Children

The Service is not intended for users under 16. We do not knowingly collect data from minors.

9. Changes

We may update this policy. Material changes will be announced via the Service dashboard.

10. Contact

Privacy questions: privacy@agent-meter.com
