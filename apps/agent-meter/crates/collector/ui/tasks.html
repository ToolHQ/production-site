Tasks

Group LLM activity by user-defined task. Useful when an agent works on a feature spanning many calls.

How to use

Wrap a unit of work to attribute cost & latency:

POST /tasks/start  {"task_id":"T-324","repo":"production-site","ide":"vscode","agent":"copilot"}
POST /tasks/end    {"task_id":"T-324"}
